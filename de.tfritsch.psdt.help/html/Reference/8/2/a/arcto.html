arcto

x1 y1 x2 y2 r arcto xt1 yt1 xt2 yt2

produces the same effect as arct. It also returns the two tangent point coordinates (xt1, yt1) and (xt2, yt2) in user space. arcto is not allowed as an element of a user path, whereas arct is allowed.

Errors: limitcheck, nocurrentpoint, stackunderflow, typecheck, undefinedresult

See Also: arc, arcn, arcto, curveto
