execstack 
array execstack subarray

stores all elements of the execution stack into array and returns an object describing the initial n-element subarray of array, where n is the current depth of the execution stack. execstack copies the topmost object into element n-1 of array and the bottommost one into element 0 of array. The execution stack itself is unchanged. If the length of array is less than the depth of the execution stack, execstack executes a rangecheck error.

Errors: invalidaccess, rangecheck, stackunderflow, typecheck

See Also: countexecstack, exec
