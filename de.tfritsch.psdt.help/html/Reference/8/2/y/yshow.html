yshow [LEVEL 2]

string numarray  yshow -
string numstring yshow -

is similar to xyshow. However, for each character shown, yshow extracts only one number from numarray or numstring. It uses that number as the y displacement and the value zero as the x displacement. In all other respects, it behaves the same as xyshow.

Errors: invalidaccess, invalidfont, nocurrentpoint, rangecheck, stackunderflow, typecheck

See Also: xyshow, show
