truncate

num1 truncate num2

truncates num1 toward zero by removing its fractional part. The type of the result is the same as the type of the operand.

Examples 

 3.2 truncate  ⇒   3.0
-4.8 truncate  ⇒  -4.0
 99  truncate  ⇒   99

Errors: stackunderflow, typecheck

See Also: ceiling, floor, round, cvi
