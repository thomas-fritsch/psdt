ISOLatin1Encoding 

- ISOLatin1Encoding array

pushes the ISO Latin-1 encoding vector on the operand stack. This is a 256-element literal array object, indexed by character codes whose values are the character names for those codes. ISOLatin1Encoding is not an operator; it is a name in systemdict associated with the array object.

Roman text fonts produced by Adobe usually use the StandardEncoding encoding vector. However, they contain all the characters needed to support the use of ISOLatin1Encoding. A font can have its Encoding array changed to ISOLatin1Encoding by means of the procedure shown in Section 5.9.1, "Changing the Encoding Vector." The contents of ISOLatin1Encoding are documented in Appendix E.

Errors: stackoverflow
