ceiling

num1 ceiling num2

returns the least integer value greater than or equal to num1. The type of the result is the same as the type of the operand.

Examples 

 3.2 ceiling  ⇒   4.0
-4.8 ceiling  ⇒  -4.0
 99 ceiling   ⇒   99

Errors: stackunderflow, typecheck

See Also: floor, round, truncate, cvi
