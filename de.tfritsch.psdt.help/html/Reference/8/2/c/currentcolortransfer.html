currentcolortransfer LEVEL 2 
- currentcolortransfer redproc greenproc blueproc grayproc

returns the current transfer functions in the graphics state for all four color components. If the current transfer functions were specified by setcolortransfer, currentcolortransfer returns those four operands. If settransfer was used, currentcolortransfer returns the single transfer function, repeated four times.

Errors: stackoverflow

See Also: setcolortransfer, settransfer
