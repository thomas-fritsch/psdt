maxlength

dict maxlength int

returns the capacity of the dictionary dict—in other words, the maximum number of entries that dict can hold using the virtual memory currently allocated to it. In LanguageLevel 1, maxlength returns the length operand of the dict operator that created the dictionary; this is the dictionary's maximum capacity (exceeding it causes a dictfull error). In a LanguageLevels 2 and 3, which permit a dictionary to grow beyond its initial capacity, maxlength returns its current capacity, a number at least as largeas that returned by the length operator.

Examples 

/mydict 5 dict def
mydict length       ⇒ 0
mydict maxlength    ⇒ 5

Errors: invalidaccess, stackunderflow, typecheck

See Also: length, dict
