undefinefont 

key undefinefont -

removes key and its associated value (a font or CIDFont dictionary) from the font directory, reversing the effect of a previous definefont operation. undefinefont is a special case of the undefineresource operator applied to the Font category. For details, see undefineresource and Section 3.9, "Named Resources."

Errors: stackunderflow, typecheck

See Also: definefont, undefineresource
