readstring

file string readstring substring bool

reads characters from file and stores them into successive elements of string until either the entire string has been filled or an end-of-file indication is encountered in file. readstring then returns the substring of string that was filled and a boolean indicating the outcome (true normally, false if end-of-file was encountered before the string was filled). If string is zero-length, a rangecheck error occurs.

All character codes are treated the same—as integers in the range 0 to 255. There are no special characters (in particular, the newline character is not treated specially). However, the communication channel may usurp certain control characters; see Section 3.8, "File Input and Output."

Errors: invalidaccess, ioerror, rangecheck, stackunderflow, typecheck

See Also: read, readhexstring, readline
