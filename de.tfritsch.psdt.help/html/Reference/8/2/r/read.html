read

file read int true  (if not end-of-file)
              false (if end-of-file)

reads the next character from the input file file, pushes it on the stack as an integer, and pushes true as an indication of success. If an end-of-file indication is encountered before a character has been read, read closes the file and returns false. If some other error indication is encountered (for example, a parity or checksum error), an ioerror occurs.

Errors: invalidaccess, ioerror, stackoverflow, stackunderflow, typecheck

See Also: readhexstring, readline, readstring, bytesavailable
