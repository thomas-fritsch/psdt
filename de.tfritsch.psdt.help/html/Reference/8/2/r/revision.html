revision 

- revision int

is an integer designating the current revision level of the product in which the PostScript interpreter is running. Each product has its own numbering system for revisions, independent of those of any other product. This is distinct from the value of version in systemdict, which is the revision level of the PostScript interpreter, which is the revision level of the PostScript interpreter, without regard to the product in which it is running.

Errors: stackoverflow

See Also: languagelevel, product, serialnumber, version
