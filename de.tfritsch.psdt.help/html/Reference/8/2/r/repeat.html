repeat

int proc repeat -

executes proc int times, where int is a nonnegative integer. This operator removes both operands from the stack before executing proc for the first time. If proc executes the exit operator, repeat terminates prematurely. repeat leaves no results of its own on the stack, but proc may do so.

Examples 

4 {(abc)} repeat               ⇒ (abc)(abc)(abc)(abc)
1 2 3 4 3 {pop} repeat         ⇒ 1     % Pops 3 values (down to the 1)
4 {} repeat                    ⇒       % Does nothing four times
mark 0 {(won't happen)} repeat ⇒ mark

In the last example, a zero repeat count meant that the procedure is not executed at all, hence the mark is still topmost on the stack.

Errors: rangecheck, stackunderflow, typecheck

See Also: for, loop, forall, exit
