Ghostscript Preferences

Interpreter

Select Window > Preferences... > Postscript > Ghostscript to open the Ghostscript preference page.

If the interpreter is not configured yet, configure it now:

For Windows:
Navigate to the Ghostscript installation directory. Select the executable (it is called "gswin32c.exe" or "gswin64c.exe").
For Linux or Unix:
Select "/usr/bin/gs".
