Debug Perspective

The Debug Perspective contains the views that you would use while debugging programs:

  • Debug View - Here you can control (stop, pause, and resume) the debugging process. You can also decide whether to step into, step over or step out of certain PostScript tokens.
  • Variables View - Displays the various variables in your PostScript process.
  • Breakpoints View - Displays the breakpoints you have entered.
  • Expressions View - Displays the progress of selected variables. The view will only be displayed if you have selected to watch a variable.
  • Editor - Displays the code at the relevant sections, according to which line is selected in the Debug View window.
  • Outline View - Displays the hierarchical structure of the PostScript code.
  • Console View - Displays any output of the Ghostcript process.
  • Tasks View - Displays tasks that were added to your files (if applicable).
